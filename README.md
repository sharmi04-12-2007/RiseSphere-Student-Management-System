# RiseSphere - AI-Enabled Smart Student Management Portal

 Project Overview

RiseSphere is a Student Management Portal developed as a Full Stack Web Development project. The system helps students, faculty, parents, and administrators manage academic activities through a centralized platform.

---

 Features

- Home Page
- Student Registration
- Student Login
- Dashboard
- Attendance Tracking
- Internal Marks
- Weekly Assignments
- Study Materials
- Timetable
- Semester GPA
- Fee Status
- AI Study Planner
- Career Readiness Score
- Notifications

---

 Technologies Used

- HTML5
- CSS3
